POLÍTICA DE PRIVACIDADE

  1. Coleta de Dados

    Durante o cadastro, o site coleta informações pessoais, como nome e endereço de e-mail, que são necessárias para gerenciar o acesso ao sistema.

  2. Armazenamento de Dados

    Os dados fornecidos pelos usuários são armazenados de forma segura no Firestore, uma plataforma de banco de dados mantida pelo Google, que segue altos padrões de segurança.

  3. Uso de Dados

    Os dados coletados são utilizados exclusivamente para o funcionamento básico do site. Não há qualquer finalidade comercial ou compartilhamento com terceiros.

  4. Segurança de Dados

    Foram implementadas medidas técnicas para proteger os dados armazenados. No entanto, é importante lembrar que nenhuma transmissão de dados pela internet é completamente segura.

  5. Solicitações dos Usuários

    Caso deseje excluir seus dados ou obter mais informações sobre o armazenamento, entre em contato pelo e-mail: willnaresse@gmail.com.

  6. Finalidade do Projeto

    Reiteramos que este site foi criado como um projeto experimental e não possui fins lucrativos.
